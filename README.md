## Ad Campaign Analytics in Power BI
End-to-End Power BI project: Python pipeline cleaning 400k ad interaction events -> Power BI star schema model -> 4-page interactive dashboard covering reach, engagement, cost efficiency, and audience intelligence.

## Tech Stack
Power BI · Python (pandas) · DAX · Power Query

## Features
16 DAX measures · TOP N · Time Intelligence (MTD/YTD/MoM) · Decomposition Tree · Key Influencers AI Visual · Drill-through · Custom tooltips · Bookmarks · Conditional formatting
